## Descripción del proyecto

Este proyecto corresponde al **Trabajo Práctico 1 del Parcial de Programación 4**, desarrollado con Angular.

La aplicación incluye un sistema de autenticación, gestión de usuarios y navegación entre vistas protegidas, aplicando conceptos de componentes, servicios, rutas y manejo de estado.



## Sprints del proyecto

### Sprint 1
- Deploy activo

  - Login
  - Registro
  - Home
  - Quién Soy
- Navegación entre vistas
